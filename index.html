SqueezeNews

The ‘SqueezeNews’ project develops open-source software for extracting and curating articles. The ‘SqueezeNews’ allows for extracting articles using python web-crawling. It is designed to easily bring up the full text and keywords of news articles. Also, it provides various visual data(e.g. word cloud, graphs) according to the frequency of keywords.
